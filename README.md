# ⚙️ .github — Configurações da Organização SI-UNINTA

Repositório especial de configuração da organização **[si-uninta](https://github.com/si-uninta)**.

## O que este repositório faz

| Arquivo / Pasta | Função |
|---|---|
| `profile/README.md` | Perfil público exibido na página da organização |
| `ISSUE_TEMPLATE/` | Templates de Issues aplicados a todos os repositórios |
| `PULL_REQUEST_TEMPLATE.md` | Checklist padrão para Pull Requests |
| `CONTRIBUTING.md` | Guia de contribuição para alunos e colaboradores |
| `CODE_OF_CONDUCT.md` | Código de conduta da comunidade acadêmica |

## Sobre a organização

Esta organização reúne os repositórios acadêmicos do curso de **Bacharelado em Sistemas de Informação — UNINTA Campus Tianguá**.

- 🌐 Site: [si-uninta.github.io](https://si-uninta.github.io)
- 🏫 Instituição: [uninta.edu.br](https://uninta.edu.br)
