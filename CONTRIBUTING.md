# 🤝 Como Contribuir

Obrigado pelo interesse em contribuir com os materiais do curso de **Sistemas de Informação — UNINTA Tianguá**!

## Quem pode contribuir?

- Alunos do curso de SI
- Professores e colaboradores da UNINTA
- Ex-alunos que queiram retribuir à comunidade

## Tipos de contribuição aceitas

- ✏️ Correção de erros em materiais (texto, código, fórmulas)
- 📎 Adição de exemplos ou exercícios resolvidos
- 🔗 Atualização de links quebrados
- 🌐 Melhorias no site do curso
- 📄 Novos templates ou modelos ABNT

## Passo a passo

1. **Fork** o repositório desejado
2. Crie uma branch descritiva:
   ```bash
   git checkout -b fix/corrige-formula-logica
   # ou
   git checkout -b feat/adiciona-exercicio-sql
   ```
3. Faça suas alterações e commit seguindo o padrão:
   ```
   fix: corrige erro na fórmula de tautologia (slide 3)
   feat: adiciona exercício resolvido de normalização
   docs: atualiza referências bibliográficas ABNT
   ```
4. Envie um **Pull Request** com descrição clara das mudanças
5. Aguarde a revisão de um mantenedor

## Padrões de qualidade

- Textos acadêmicos devem seguir **ABNT NBR 6022/6023**
- Código deve ter comentários em **português**
- Nomes de arquivos em **kebab-case** e sem acentos: `logica-proposicional.pdf`
- Imagens em `/assets/` dentro do repositório correspondente

## Dúvidas

Abra uma [Issue](https://github.com/si-uninta/.github/issues) com a tag `pergunta`.
